# Maná en Casa

PWA (Progressive Web App) desarrollada para el **Centro Médico Integrativo Maná**, orientada a brindar acompañamiento integrativo a pacientes de manera digital, accesible y personalizada.

---

## 🌐 Acceso a la aplicación

🔗 https://app.centromedicointegrativomana.com

---

## 🚀 Estado actual del proyecto

* Aplicación desarrollada en **React + Vite**
* Desplegada en **Vercel**
* Configurada como **PWA instalable**
* Dominio personalizado activo
* Navegación funcional entre módulos
* Botón de contacto directo vía WhatsApp
* Experiencia tipo app móvil

---

## 🎯 Objetivo del proyecto

Convertir esta aplicación en una **plataforma exclusiva para pacientes**, que permita:

* Acceso privado por usuario
* Contenido personalizado según cada paciente
* Seguimiento de procesos de bienestar
* Comunicación directa con el equipo médico
* Escalabilidad hacia modelos clínicos apoyados en IA

---

## 🧠 Próxima fase (en desarrollo)

Implementación de acceso restringido:

* Sistema de autenticación (login)
* Protección de rutas
* Visualización de contenido según usuario
* Preparación para integración con base de datos

---

## 🛠️ Tecnologías utilizadas

* React

## 📱 Características clave
